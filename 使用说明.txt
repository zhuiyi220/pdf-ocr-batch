PDF OCR 批量处理工具 - 使用说明
=====================================

✓ 所有环境已配置完成，可直接使用

运行方式：
---------
方式1: 双击 run.bat 启动
方式2: 命令行运行 python pdf_ocr_batch.py

使用步骤：
---------
1. 运行程序后会弹出文件夹选择窗口
2. 第一步：选择包含PDF文件的输入文件夹
3. 第二步：选择输出文件夹（处理后的PDF将保存在这里）
4. 程序自动处理所有PDF文件
5. 处理完成后会显示成功和失败的数量

功能特点：
---------
✓ 保留原始PDF的所有格式、布局和图片
✓ 支持中英文混合识别
✓ 批量处理多个文件
✓ 多线程并行处理（2个线程）
✓ 实时显示处理进度

已安装组件：
-----------
✓ Tesseract OCR 5.3.3
✓ 中文简体语言包 (chi_sim)
✓ 英文语言包 (eng)
✓ Python依赖包 ocrmypdf

测试目录：
---------
test_input/  - 可将测试PDF放入此文件夹
test_output/ - 处理后的PDF将输出到此文件夹
